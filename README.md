# 📞 Vishing Awareness Simulator

## 📌 Project Overview

The **Vishing Awareness Simulator** is a Python-based educational project designed to demonstrate how voice phishing (vishing) attacks work. It helps users recognize common scam tactics and promotes cybersecurity awareness through an interactive command-line simulation.

This project was completed as part of my **Cyber Security Internship at SQROCK IT SOLUTION**.

---

## 🎯 Objectives

- Understand the concept of Vishing (Voice Phishing).
- Identify common warning signs of scam calls.
- Raise cybersecurity awareness.
- Practice Python programming through a real-world security scenario.

---

## ✨ Features

- Interactive command-line simulation
- Demonstrates common vishing techniques
- Displays security warnings and awareness tips
- Beginner-friendly Python implementation
- Easy to run and understand

---

## 🛠️ Technologies Used

- Python 3
- Command Line Interface (CLI)
